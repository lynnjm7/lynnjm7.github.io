An Intro to Swift's Type System

Josh Lynn

Abstract

Publication
Las Vegas iOS Developers Meetup
Date
October, 2016
Links
Slides Code

This talk was presented at the Las Vegas iOS Developers Meetup and presented an introduction to the basics of the Swift type system and how to describe basics data structures in the Swift programming language.

This talk begins with a treatment of basic data oriented development and continues on a tour through the basic structure of defining data in Swift. This talk was prepared as part of a series of talks on helping new iOS Developers learn Swift and programming.
